# air-quality-respiratory-health_using_R

# Air Quality and Respiratory Health: A Statistical Analysis

This project explores the relationship between air pollution levels and the incidence of respiratory diseases in U.S. cities between 2015 and 2020. It uses publicly available health and environmental data to analyze patterns, correlations, and trends using R.

## Skills Demonstrated
- Data cleaning and wrangling
- Descriptive statistics and visualization
- Linear regression and interpretation
- Communicating data-driven insights

## Tools
- R
- ggplot2
- dplyr
- readr
- lubridate

## Data Sources
- EPA Air Quality Data
- CDC Respiratory Illness Reports
